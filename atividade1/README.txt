Como compilar:

Primeiramente, certifique - se de que o diretório atual é o diretório que contém a pasta atividade1. Uma vez que esteja no diretório correto, execute os seguintes comandos:

    javac atividade1/TCPClient.java
    javac atividade1/TCPServer.java

Como executar:

Para executar o servidor, execute o seguinte comando:

    java atividade1.TCPServer

Para executar o cliente, abra um novo terminal e execute o seguinte comando:

    java atividade1.TCPClient
