Atividade 1:

    -  Como compilar:
        - javac UDPChat.java

    -  Como executar:
        - java UDPChat.java
    
    - Bibliotecas usadas:

        - java.io.*;
        - java.net.*;
        - java.util.Scanner;
        - javax.swing.JOptionPane;

    Exemplo de uso:
        - Usuário digita o IP, a porta de destino e seu apelido
        - Envia mensagens seguindo o padrão <tipo_mensagem>:<mensagem>

        - Para mensagens normais
            1:<mensagem>

        - Para emoji
            2:<codigo_do_emoji>

        - Para URL
            3:<url>

        - Para ECHO
             4:<mensagem>


Atividade 2:

Antes de realizar os comandos abaixo, é necessário estar no diretório onde se encontra a pasta "UDP".

    -  Como compilar:
        -   javac  UDP/UDPClient.java
        -   javac  UDP/UDPServer.java

    -  Como executar:
        -   java UDP.UDPClient
        -   java UDP.UDPServer

    - Bibliotecas usadas:
        
        - java.net.*;
        - java.io.*;
        - java.nio.file.Files;
        - java.security.MessageDigest
        - java.util.Arrays;

    Exemplo de uso:
        - Cliente:
            - Digite o nome do arquivo (ex: arquivo.txt):
            - arquivo.txt
